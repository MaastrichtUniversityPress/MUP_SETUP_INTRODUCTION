# Understanding Your Repository

When MUP sets up your book, they create a **repository** for you on GitHub. Think of a repository as collaborative cloud drive, for example google drive. It simply contains all the necessery files that make up your book. But as previously mentioned, you will not have to set it up, therefore, all you will have to learn is how to use it. This section is a guide on where you can find necessery files, how to navigate this new landscape and what each part does. 

## What's in your repository?

Your repository contains three types of things:

**1. Your content files ('.md'files)**  
These files is where you would actually write your book. It would be written in a format called Markdown, this could be familiar to you if you have used note taking applications, like Notion or Obsidian. Each seperate '.md' file becomes a seperate page on the website, for example 'intro.md' would be the first page that would pop up on your readers screen. 

**2. The table of contents ('_toc.yml')**  
To organize your books structure, the '_toc.yml' file will be used to determine which pages will appear, in what order, and how they are grouped into chapters. This page is a bit more syntax heavy, therefore it will be thoroughly explained on how to edit it in Chapter 3. 

**3. The configuration file ('_config.yml')**  
This file contains the general settings for your book, such as the title, authors name, and logo. This will mostly be filled in for you by MUP, however you may need to update it occasionally. 

## What do you actually need to edit?

As an author, you will spend almost all of your time editing your 
`.md` content files. The table of contents may need occasional updates 
when you add or remove pages. The configuration file you can largely 
leave alone.

```{tip}
Not sure where to start? Head to your repository on GitHub, try to find the three things that were mentioned: 'intro.md', '_toc.yml', and '_config.yml'. Then open 'intro.md', and click the pencil icon to start editing, and try to make some minor changes!
```
